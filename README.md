# moodLight
Audio Equalizer with Neopixel on Arduino 


# 부품 스펙
## 피에죠
* 3.3V
## LM393
* 3.3V~5V
## Neopixel
* 5V
## OLED
* 3.3V

# 회로구성
* A0 - 피에조
* A1 - 소리감지
* A4 - OLED SDA
* A5 - OLED SCL
* D6(output) - neopixel input
